## 📌 Descrição
Projeto desenvolvido como parte do desafio da DIO, com o objetivo de aplicar conceitos de desenvolvimento web utilizando HTML, CSS e JavaScript.

## 🛠️ Tecnologias
- HTML5
- CSS3
- JavaScript
- Git e GitHub

## ⚙️ Funcionalidades

- Interação com botão
- Manipulação do DOM
